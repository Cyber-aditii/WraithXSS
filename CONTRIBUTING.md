# Contributing to XSS Param Hunter

Thank you for your interest in improving XSS Param Hunter! We welcome contributions from the community.

## How to Contribute

1.  **Report Bugs**: Open an issue describing the bug and how to reproduce it.
2.  **Suggest Enhancements**: Open an issue with a clear description of the feature request.
3.  **Submit Pull Requests**:
    *   Fork the repository.
    *   Create a new branch for your feature or bug fix.
    *   Make your changes and ensure the code follows the project's style.
    *   Submit a pull request with a detailed description of your changes.

## Development Setup

1.  Clone the repository.
2.  Install dependencies: `pip install -r requirements.txt`.
3.  Ensure you have the required external tools: `subfinder`, `httpx`, `gau`.

## Ethical Guidelines

This tool is for educational and authorized security testing purposes only. Do not use it on targets without explicit permission.
